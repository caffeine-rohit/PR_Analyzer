# GitHub PR Analyzer
   
   A modern web app to analyze GitHub Pull Requests with smart insights.
   
   ## Features
   - 🚀 Instant PR analysis
   - 📊 Smart file categorization
   - 🔍 Potential issue detection
   - ⚡ Review time estimation
   
   ## Tech Stack
   - React + Vite
   - Tailwind CSS
   - GitHub REST API
   
   ## Setup
```bash
   npm install
   npm run dev
```
